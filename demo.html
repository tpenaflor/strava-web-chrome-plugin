🚀 Extension Features

  • Automatically detects your own Strava activities
  • Downloads and parses FIT files for detailed metrics
  • Supports multiple LLM providers (OpenAI, Anthropic, Local)
  • Provides comprehensive AI-powered performance analysis
  • Offers training insights and recommendations
  • Analyzes pacing, zones, and effort distribution
  • Secure local processing with configurable API keys
  • Works with cycling, running, and other endurance activities

🛠️ Installation Instructions

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked" and select the extension folder
  4. Configure your LLM provider in the extension options
  5. Visit any of your Strava activities to see the analyze button!
